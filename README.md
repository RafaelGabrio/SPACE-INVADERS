<h2 id="started">ğŸš€ Getting started</h2>

Run with the _Live Server_ extension in VS Code.

<h3>Prerequisites</h3>

- [Live Server](https://marketplace.visualstudio.com/items?itemName=yandeu.five-server)
- Your Browser

<h3>Cloning</h3>

```bash
git clone https://github.com/RafaelGabrio/SPACE-INVADERS.git
```

<h3>Starting</h3>

- Click *Go Live* in the Status Bar (bottom of VSCode)
- _or_ Right-Click a Open File > *Open with Five Server*
- _or_ Right-Click a File in the Explorer > *Open with Five Server*
- _or_ Right-Click a Folder in the Explorer > *Open with Five Server (root)*
